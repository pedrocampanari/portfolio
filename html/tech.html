TECNOLOGIAS

Tools & Hobbies

    • HTML

      HTML (HyperText Markup Language): Linguagem de marcação fundamental para criar e estruturar páginas web, definindo o conteúdo e a estrutura de sites.

    • CSS

      CSS (Cascading Style Sheets): Linguagem de estilo usada para descrever a apresentação e o layout de documentos HTML, permitindo a personalização de aparência e design de páginas web.

    • JavaScript

      JavaScript (JS): Linguagem de programação amplamente utilizada para criar interatividade e funcionalidades dinâmicas em páginas web, como animações, validação de formulários e interações com o usuário.

    • Bootstrap

      Bootstrap: Framework front-end que facilita o desenvolvimento de sites responsivos e móveis, oferecendo um conjunto de ferramentas e componentes prontos para usar com HTML, CSS e JavaScript.

    • AWS

      AWS (Amazon Web Services): Plataforma de serviços de computação em nuvem que oferece uma ampla gama de serviços como armazenamento, processamento e análise de dados, escalabilidade e gerenciamento de recursos.

    • GitHub

      GitHub: Plataforma de hospedagem de código-fonte que utiliza o sistema de controle de versão Git, permitindo colaboração, gerenciamento de projetos e versionamento de código em projetos de software.

    • MongoDB

      MongoDB: Banco de dados NoSQL orientado a documentos, que armazena dados em formato JSON-like e é ideal para aplicações que necessitam de flexibilidade e escalabilidade.

    • C++

      C++: Linguagem de programação poderosa e de alto desempenho, utilizada para desenvolvimento de sistemas, aplicativos de software, e jogos, conhecida por suas capacidades de manipulação de recursos e eficiência.

    • Arduino IDE

      Arduino IDE: Ambiente de desenvolvimento integrado para programar microcontroladores Arduino, facilitando a criação de projetos de eletrônica e prototipagem rápida com código C/C++.

    • Python

      Python: Linguagem de programação de alto nível conhecida por sua simplicidade e legibilidade, amplamente utilizada em desenvolvimento web, ciência de dados, automação, e inteligência artificial.

    • WordPress

      WordPress: Sistema de gerenciamento de conteúdo (CMS) que permite criar e gerenciar sites e blogs com facilidade, oferecendo uma ampla gama de temas e plugins para personalização.

    • Power BI

      Power BI: Ferramenta de análise de dados e visualização da Microsoft que permite criar relatórios e dashboards interativos, facilitando a análise e interpretação de dados empresariais.

    • Git

      Git: Sistema de controle de versão distribuído que gerencia e rastreia mudanças em código-fonte durante o desenvolvimento de software, promovendo a colaboração e o versionamento eficiente.

    • Node.js

      Node.js: Ambiente de execução JavaScript no lado do servidor, que permite desenvolver aplicações de rede escaláveis e de alto desempenho utilizando JavaScript fora do navegador.

    • Photoshop

      Photoshop: Software de edição de imagens e design gráfico desenvolvido pela Adobe, amplamente utilizado para manipulação de fotos, criação de gráficos e design visual.

    • Premiere Pro

      Premiere Pro: Software de edição de vídeo da Adobe, utilizado para edição e produção de vídeos profissionais, oferecendo uma gama de ferramentas avançadas para montagem e pós-produção.

    • Illustrator

      Illustrator: Ferramenta de design gráfico vetorial da Adobe, ideal para criação de ilustrações, logotipos, e gráficos escaláveis com alta qualidade e precisão.

    • Animate

      Animate: Software da Adobe para criar animações vetoriais interativas e animações em HTML5, utilizado para design e desenvolvimento de conteúdo animado para a web.

    • Adobe XD

      Adobe XD: Ferramenta de design e prototipagem de interfaces da Adobe, que permite criar e testar protótipos interativos de aplicativos e sites com foco na experiência do usuário.

    • Audition

      Audition: Software de edição de áudio da Adobe, utilizado para gravação, edição e mixagem de áudio profissional, oferecendo uma gama de ferramentas para produção sonora e pós-produção.
